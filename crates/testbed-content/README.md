# Advanced testbed content

This crate intentionally uses the advanced `game-kit` surface for manual
systems, RON maps, patrol setup, and lower-level content experiments.

Do not copy this crate when starting a beginner game. Copy
`examples/one-file-demo`, `examples/no-rust-shapes-demo`,
`examples/script-like-custom-rules`, `simple-content`, or
`templates/simple-demo` first. `arena-content` is the later structured
beginner-Rust step.
