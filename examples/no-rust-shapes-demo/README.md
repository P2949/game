# No Rust Shapes Demo

This is the final standalone beginner acceptance example. It uses only
`game_starter::prelude::*` and named game concepts: assets, maps, actors, rules,
scenes, UI, audio, pickups, projectiles, triggers, doors, drops, and checkpoints.

Run it with:

```bash
cargo run -p no-rust-shapes-demo
```

Press Space or click Start, move with WASD or the arrow keys, shoot with Space,
and press F5 after editing `assets/maps/game.txt` in a debug build.
